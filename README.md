# Рыбный остров — учебный веб-проект

## Структура

- `register.htm` — регистрация пользователя
- `login.htm` — авторизация
- `index.htm` — оформление заказа и просмотр ассортимента
- `reports.htm` — отчётность по продажам
- `admin.htm` — административная панель
- `css/styles.css` — общий стиль сайта
- `js/main.js` — простая логика для выбора товаров на главной странице
- `img/` — скриншоты страниц

## Как открыть локально

1. Скачайте папку проекта.
2. Откройте файл `index.htm` в браузере.
3. Для редактирования откройте папку проекта в VS Code.
